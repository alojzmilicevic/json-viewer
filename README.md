# JSON Parser 📊
Welcome to JSON Parser, a web application built with React, TypeScript, and Material UI that allows you to easily parse and visualize JSON data. With our advanced tokenization and color-coding system, you can quickly identify different JSON elements and collapse parts of the tree to focus on specific sections.

# Features 🎨
- Tokenization of JSON data with color-coded highlighting for different elements
- Collapsible tree view for easier navigation of large JSON documents
- User-friendly interface for inputting and displaying JSON data

# Getting Started 🚀
To get started with JSON Parser, follow these steps:

- Clone or download the repository
- Navigate to the project directory
- Install the dependencies by running npm install
- Start the development server by running npm start

# Tech Stack 🛠
JSON Parser is built with the following technologies:

- React
- TypeScript
- Material UI
- HTML/CSS
- Node.js
- Next.js just to learn a little about it, no server really...
